# preserve_on_blur
**Plugin for Selectize.js**

Include the JS file:

```
<script type="text/javascript" src=".../selectize.js/plugins/preserve_on_blur.js"></script>
```

Define the plugin in your constructor:

```
$( '#selector').selectize( {    
    ...
    plugins: [ 'preserve_on_blur' ],           
    ...
} );
```
